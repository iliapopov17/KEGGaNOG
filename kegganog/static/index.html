What is KEGGaNOG?

KEGGaNOG takes eggNOG-mapper v2 annotation files and computes KEGG pathway completeness scores, producing publication-ready visualizations.

Input files: .emapper.annotations (tab-separated output of eggNOG-mapper v2). One file = one sample.

Single mode — one sample → heatmap + barplot.
Multi mode — multiple samples → heatmap, radarplot, correlation network, stacked bar, streamgraph.
